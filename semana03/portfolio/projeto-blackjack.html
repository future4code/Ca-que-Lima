Blackjack


Este projeto foi o meu favorito até agora. O objetivo era criar um jogo de Blackjack em que o usuário jogava contra o computador. Ao iniciar uma partida, o jogador via suas duas cartas e apenas uma do computador. Então, uma decisão tinha que ser tomada: comprar ou não mais uma carta, para tentar chegar mais perto de 21 pontos. Caso quisesse comprar uma carta, e passasse de 21 pontos, o computador ganhava imediatamente, e caso não passasse, era perguntado se queria novamente comprar mais uma carta. Escolhendo para de comprar cartas, era a vez do computador comparar cartas para tentar ganhar. Se o computador conseguisse mais pontos que o usuário e menos que 21, ele ganhava, se não, o usuário ganhava. Após tudo isso, o resultado era mostrado.
